Minha pesquisa sobre HTML e CSS

Oque é HTML?

Ela é uma linguagem de marcação e ela serve como "esqueleto" para qualquer página da web.

O que é CSS?

O CSS é a sigla para Cascading Style Sheets. Ele é responsável pela apresentação visual da página.

Importância do Github

A importância do GitHub para programadores é que ele serve como a espinha dorsal do desenvolvimento de software moderno, sendo indispensável por três motivos principais. Primeiro, ele oferece controle de versão ao hospedar o sistema Git na nuvem, garantindo a segurança do código ao registrar cada alteração e permitindo o retorno a versões anteriores, evitando perdas e erros. Segundo, ele é a principal ferramenta de colaboração global, facilitando o trabalho em equipe através de funcionalidades como os Pull Requests, que padronizam a revisão e a integração de código, assegurando a qualidade do projeto. Terceiro, o GitHub atua como um portfólio profissional essencial, onde os programadores exibem suas habilidades e contribuições para a comunidade e para recrutadores.
